Battle Rules

  • Both boards run at the same time, like a versus puzzle game.
  • Merges create attack power and send black garbage blocks to the opponent.
  • Garbage blocks cannot merge. They show remaining seconds and fade out.
  • Knockout: a player loses when no legal move remains.
  • Score Limit: when time ends, the higher score wins.

CPU

The CPU board is visible and moves on its own at a fixed rhythm. It picks moves by merge score, combo count, and empty cells.

Friend

This prototype supports local simultaneous controls. Online friend battles can reuse the same two-board state and replace P2 input with WebSocket messages.
